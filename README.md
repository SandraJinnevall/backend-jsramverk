# jsramverk-backend


## Installation

1. Download this code. 
2. Install all dependencies with
```
npm install
```

3. Create a config.json file and add:
```
{
    "username": "YOUR USERNAME",
    "password": "YOUR PASSWORD"
}
```
4. Run
```
npm start
```

## Routes
There are four routes, GET, POST, PUT, DELETE
<br>
**GET**: "/"
<br>
**PUT**: "/:id"
<br>
**POST**: "/"
<br>
**DELETE**: "/:id"
<br>
